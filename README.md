# Big Integer Library in C++

A modular arbitrary-precision integer library implemented with object-oriented design, operator overloading, and a compact vector-based internal representation.

## Features
- Arbitrary-precision addition, subtraction, multiplication, division, modulo, and exponentiation
- Integer square root with binary search
- Support for combinatorial helpers such as factorial, Fibonacci, and Catalan numbers
- Stream-friendly output through overloaded operators

## Build and Run
```bash
make
make run
```

## Example
The demo program exercises arithmetic, power, square root, factorial, Fibonacci, and Catalan computations.
